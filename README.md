# 💸 Expense Tracker

A simple and clean Expense Tracker web app built with **React.js**. Track your daily expenses and income, view your balance, and manage transactions — all saved locally in your browser.

## ✨ Features

- Add expense and income transactions
- View total Balance, Expense, and Budget at a glance
- Search transactions by name
- Remove any transaction
- Data persists using localStorage (no backend needed)
- Clean and minimal UI

## 🛠️ Tech Stack

- React.js (Create React App)
- React Context API for state management
- useReducer hook
- localStorage for data persistence
- Plain CSS (no external UI library)
